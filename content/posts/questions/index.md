## Question Words

Here are some of the question words in Turkish:

| Turkish   | English Meaning                |
| --------- | ------------------------------ |
| Ne?       | What?                          |
| Neden?    | Why?                           |
| Nerede?   | Where?                         |
| Ne zaman? | When? What time?               |
| Nasıl?    | How?                           |
| Hangi?    | Which?                         |
| Kim?      | Who?                           |
| Kaç?      | How many / much?   (countable) |

## Phrases

- **Ne kadar?** How much?
- **Kaça?** How much? (price when buying/selling)
- **Kaç tane?** How many?
- **Nasılsın?** How are you?
- **Neredesin?** Where are you?
- **Bu kim?** Who is this?
